# Video Conferencing Application 🎥

A real-time video conferencing application that allows users to seamlessly connect, communicate, and collaborate via video, audio, and text chat. 

## ✨ Features

* **Real-time Video & Audio:** Low-latency communication for smooth meetings.
* **Mute/Unmute & Camera Toggle:** Users have full control over their media streams.
* **Screen Sharing:** Present documents, slides, or your entire screen with a single click.
* **In-Meeting Chat:** Real-time text messaging alongside the video call.
* **Room-based Architecture:** Secure, isolated meeting rooms using unique IDs.
* **Responsive Design:** Optimized for both desktop and mobile web browsers.
